安装步骤
============================================

安装前确保你已经安装好git工具，并能够访问http://gitlab.isumeng.com

    git clone git@gitlab.isumeng.com:icms.git

* 1.安装ruby-1.9.3

* 2.安装rails-3.2.14

* 3.安装refinerycms包

* 4.安装imagemagic库

* 5.安装所有依赖的gem库
    bundle install

* 6.初始化数据库
    rake db:setup

或者分步运行如下三条命令:
    rake db:create #创建数据库
    rake db:migrate #迁移数据
    rake db:seed #对数据库植入种子数据

想了解项目支持那些批任务吗？运行<tt>rake -T</tt>可以看到所有支持的批处理任务
